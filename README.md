# Personal Website

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```

The static files will be generated in the `out` directory.

## Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions when changes are pushed to the `master` branch.

### Manual Deployment Steps (if needed):

1. Build the site: `npm run build`
2. The `out` folder contains the static files
3. GitHub Actions will automatically deploy these files to GitHub Pages

### GitHub Pages Settings:

1. Go to your repository settings
2. Navigate to "Pages" section
3. Under "Source", select "GitHub Actions"
4. The site will be available at `https://jesusCaraball0.github.io`
